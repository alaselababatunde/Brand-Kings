## Installation & Usage

### Local Development

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd Brand-Kings
   ```

2. **Open in browser**:
   - Simply open `index.html` in any modern web browser
   - No build process required - it's a static website

3. **Local server (optional)**:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   
   # Node.js (with http-server)
   npx http-server
   ```

### Browser Support

- **Modern Browsers**: Chrome 60+, Firefox 55+, Safari 12+, Edge 79+
- **Mobile**: iOS Safari 12+, Chrome Android 60+
- **Fallbacks**: Graceful degradation for older browsers

## Customization

### Color Scheme
Edit CSS variables in `styles.css`:
```css
:root {
    --deep-purple: #6a0dad;
    --gold: #ffd700;
    --primary-dark: #0f0f13;
    --secondary-dark: #1a1a20;
}
```

### Content Updates
- Update text content in HTML files
- Replace images in the project directory
- Modify form actions in `script.js`

### Adding New Pages
1. Create new HTML file following the existing structure
2. Add navigation links to all existing pages
3. Include required CSS and JavaScript files
4. Update the sitemap if needed
